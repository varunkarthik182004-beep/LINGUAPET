# LinguaPet 🐾

LinguaPet is a language learning mobile app built using Flutter. It helps users learn Tamil through structured sections and pronunciation support.

## 🚀 Features

- Beginner learning path
- Word learning with Tamil pronunciation (TTS)
- Clean UI with Flutter
- Section-based navigation

## 🧠 Tech Stack

- Flutter (Dart)
- SharedPreferences
- Text-to-Speech (TTS)
